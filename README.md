## Sobre o Projeto
O objetivo principal foi criar um portfólio online funcional utilizando tecnologias web fundamentais (HTML, CSS e JS), sem o auxílio de frameworks. O site segue o modelo de **Single Page Website** com navegação por âncoras.

### Requisitos Implementados:
* **Estrutura Semântica**: Utilização de HTML5 para a organização do conteúdo.
* **Estilização Pura**: CSS3 utilizado para criar um layout com estética Glassmorphism, sem o uso de Bootstrap ou outros frameworks de CSS.
* **Funcionalidades JavaScript**: Lógica para alternância de tema (claro/escuro) e validação obrigatória do formulário de contato.
* **Seções Obrigatórias**: Sobre mim, Formação, Portfólio e Contato.
* **Comentários**: O código-fonte contém explicações sobre os principais blocos de lógica e estrutura.

## Tecnologias Utilizadas

* JavaScript (Vanilla JS)
